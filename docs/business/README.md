## Quick Reference

### Key Metrics
- Average commission: % of refund amount
- Processing time: ~1 hour per case (experienced)
- Refund eligibility: Up to 6 years back

### Common Client Scenarios
1. Incorrect employer tax calculations
2. Multiple workplaces without tax coordination
3. Mid-year job changes
4. Unaware of entitled credits/deductions
